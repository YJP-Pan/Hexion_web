Microsoft Defender XDR

Microsoft
Defender XDR

エンドポイント・メール・ID・クラウドアプリにわたる統合 XDR 保護 — AI 調査で脅威の逃げ場をなくす。

Microsoft Defender XDR は Defender for Endpoint・Defender for Office 365・Defender for Identity・Defender for Cloud Apps を1つのコンソールに統合 — クロスドメイン攻撃チェーン相関分析・自動インシデント応答・MTTD と MTTR の大幅短縮を実現します。

Core Capabilities

Defender XDR 統合保護

Endpoint Detection & Response (MDE)

Microsoft Defender for Endpoint が Windows・macOS・Linux・iOS・Android 全体で行動検出・脆弱性管理・デバイス隔離・EDR 調査を提供します。

Email & Collaboration Protection (MDO)

Defender for Office 365 がフィッシング・悪意ある添付ファイル・BEC 詐欺をブロック — Safe Links・Safe Attachments・攻撃シミュレーションを内蔵。

Identity Protection (MDI)

Defender for Identity が Active Directory と Entra ID の異常行動を監視 — 横断移動・Pass-the-Hash・特権アカウント悪用を検出します。

Cloud App Security (MDCA)

Defender for Cloud Apps（旧 MCAS）が SaaS 可視化・シャドー IT 検出・OAuth アプリ制御・クラウドサービス全体のデータ損失防止（DLP）を提供します。

AI Automated Investigation & Response

Security Copilot AI がマルチドメインアラートを統合インシデントへ自動相関し、修復アクション（デバイス隔離・トークン失効・ファイルブロック）を実行 — MTTR を劇的に短縮します。

Attack Chain Correlation & Threat Hunting

クロスドメイン攻撃チェーン可視化が完全な侵害経路を表示。KQL による Advanced Hunting でセキュリティチームがすべてのテレメトリ上の潜在的脅威を積極的に探索できます。
