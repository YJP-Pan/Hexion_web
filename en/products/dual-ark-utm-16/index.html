Ark-UTM 16
High-Availability Dual-Unit UTM

Enterprise and IoT network security solution with HA high-availability redundancy

Ark-UTM 16 uses LIONIC DPI technology with integrated antivirus, intrusion detection, malicious website blocking, geo-blocking, and firewall. Supports high-availability (HA) dual-unit redundancy with failover under 30ms, maintaining 99.99% connection availability. Supports SSL/TLS encrypted traffic scanning and VLAN network segmentation, with CMS centralized management.

Core Features

Ark-UTM 16 Protection Features

HA Dual-Unit Redundancy

Supports high-availability (HA) redundancy — the standby unit automatically takes over upon failure, with switchover under 30ms to keep network connections uninterrupted.

SSL/TLS Encrypted Traffic Scanning

Supports SSL/TLS traffic decryption and scanning for comprehensive protection while browsing HTTPS sites, with customizable whitelists.

LIONIC DPI Engine

Patented deep packet inspection technology that identifies viruses, hacking attempts, and malicious sites, blocking them in real time.

VLAN Network Segmentation

Supports VLAN network segmentation for flexible definition of different security zones, with 2 WAN interfaces and 5 LAN interfaces per unit for versatile deployment.

Geo-Blocking

Block connections from specific countries or regions based on threat origin to effectively reduce the attack surface.

CMS Centralized Management

Monitor and manage multiple UTM 16 units simultaneously via the CMS central management system — remote configuration, firmware updates, and automated daily reports.
