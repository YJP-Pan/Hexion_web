Tera-UTM 12
LIONIC DPI POWERED

Industrial-grade multi-layer network security filter — ICS/OT protocol detection with Hardware Bypass to keep production lines uninterrupted

Tera-UTM 12 is purpose-built for industrial and OT environments, equipped with the LIONIC DPI engine for 890 Mbps protection throughput. It supports 1×GbE WAN + 1×GbE LAN network interfaces, integrates ICS/OT protocol detection (Modbus, DNP3), and includes Hardware Bypass to ensure industrial control systems remain unaffected.

Core Features

Tera-UTM 12 Protection Features

LIONIC DPI Engine

Industry-leading deep packet inspection that accurately identifies network application protocols and encrypted traffic, providing a high-performance threat analysis foundation.

High-Performance IPS

Intrusion prevention system that detects and blocks network attacks in real time without impacting industrial control system network performance.

Multi-Interface Support

1×GbE WAN + 1×GbE LAN network interfaces, suitable for bridge-mode deployment in industrial control systems without altering the existing network topology.

SSL/TLS Decryption

Deep inspection of encrypted traffic — decrypts and scans HTTPS and TLS flows to eliminate all encryption blind spots.

Application-Layer Control

Identification and control of 1,400+ applications — define granular access policies by user, group, and time schedule to enhance security and productivity.

Hardware Bypass

Supports Hardware Bypass — automatically maintains direct network pass-through during device maintenance or power loss, ensuring industrial control systems are not affected.
